1. AsyncStorage 구현
    1) 유저 로그인 시 해당 유저 모든 기록 값 로컬로 저장 => 오프라인 사용 가능 및 앱 성능 개선
    2) 유저 로그인 내역 및 해당 유저 기록 값 로드
    3) 앱 시작 시 await으로 로컬으로부터 데이터 로드
    4) 따라서 기록 사항 변동 또는 추가 시 서버로 async 동기화 및 로컬 저장소 업데이트
    5) 클라이언트 온라인일 시 주기적으로 데이터 업데이트 (혹시 모를 오류 대비)

2. AppState 기능
    1) AppState 사용으로 백그라운드 시에도 타이머 값 저장을 위해 사용
    2) 앱 종료 시점의 시간과 다시 실행 시점의 시간 차를 이용해 구현
    3) 추가로 AddLive 페이지에서 유닛 수정 시 시간 기록 후 기록 차를 이용해 시간 계산
    4) DB에 시작 시간, 마지막 유닛 수정 시간 기록

3. 스플래시 이미지
    1) 스플래시 이미지로 앱 로딩 시 사용자 경험 개선
